# HR Attrition Analysis Dashboard (Excel)

## Project Overview
This project analyzes employee attrition using a Human Resources dataset containing **1,470 employee records and 40+ HR-related variables**. The goal of this project is to identify patterns in employee turnover and generate insights that can help organizations improve employee retention.

The analysis was performed using **Microsoft Excel**, where the dataset was cleaned, transformed, and analyzed using Pivot Tables, charts, and data visualization techniques. An **interactive HR analytics dashboard** was developed to explore attrition trends across departments, job roles, and demographic groups.

---

## Project Objectives
The main objectives of this project are:

- Analyze employee attrition patterns across different workforce dimensions  
- Identify factors influencing employee turnover  
- Develop key HR performance indicators (KPIs)  
- Build an interactive dashboard to support HR decision-making  

---

## Dataset Information
The dataset consists of **1,470 employee records** with more than **40 attributes** related to employee demographics, job characteristics, and satisfaction levels.

- Job Role  
- Gender

- Marital Status  
- Job Satisfaction

- Overtime

- Attrition Status

## Data Analysis Process
The analysis followed these steps:

1. Data cleaning and formatting of the HR dataset  
2. Categorizing employee attributes such as age groups and attrition status  
3. Creating pivot tables to analyze workforce data  
4. Calculating key HR metrics including attrition rate and employee count  
5. Developing visual charts to represent workforce patterns  
6. Designing an interactive HR analytics dashboard using slicers  

---

## Dashboard Features
The HR analytics dashboard includes:

- Total Employee Count  
- Attrition Count  
- Attrition Rate  
- Attrition by Department  
- Attrition by Age Group  
- Attrition by Job Role  
- Attrition by Education Field  

Interactive **slicers enable dynamic filtering** for better data exploration and analysis.

---

## Key Insights
Some insights derived from the analysis include:

- Certain departments experience **higher employee attrition rates**  
- **Younger employees** show relatively higher turnover patterns  
- Employees working **overtime** tend to have higher attrition probability  
- Attrition trends vary across **job roles and education fields**  

---

## Business Recommendations
Based on the analysis, organizations may consider the following actions:

- Improve employee engagement programs in high-attrition departments  
- Monitor overtime workloads to reduce employee burnout  
- Implement retention strategies for early-career employees  
- Conduct regular employee satisfaction surveys
